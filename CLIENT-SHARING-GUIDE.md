# 📱 クライアントさんへの閲覧専用モード共有ガイド

## 🎯 最適な方法：VercelのURLをLINEで送る

### 1. VercelのURLを確認

1. Vercelダッシュボード（https://vercel.com/dashboard）を開く
2. スカウト管理アプリのプロジェクトを選択
3. 「Settings」→「Domains」でカスタムドメインを設定（オプション）
4. または、デフォルトのVercel URLを使用：
   - 例: `https://scout-manager-xxxxx.vercel.app`

### 2. LINEで送るメッセージ例

```
【スカウト候補者管理アプリ】

こちらのURLからアクセスできます：
https://your-vercel-url.vercel.app

📋 使い方：
・候補者カードをクリックして詳細を確認
・ステータスを変更（連絡する、保留、NG）
・メモを追加・編集
・「レビュー完了」ボタンで連絡

⚠️ 閲覧専用モードのため、新規登録はできません。
```

### 3. より見やすくする方法

LINEのメッセージに以下のように送る：

```
━━━━━━━━━━━━━━━━━━━━

━━━━━━━━━━━━━━━━━━━━

https://your-vercel-url.vercel.app
```

## 🔒 セキュリティについて

### オプション1: パスワード保護（推奨）

Vercelでパスワード保護を設定：
1. Vercelダッシュボード → プロジェクト → Settings
2. 「Deployment Protection」を有効化
3. パスワードを設定
4. クライアントさんにパスワードも一緒に送る

### オプション2: カスタムドメイン + 認証

将来的に認証機能を追加することも可能です。

## 📝 送る前に確認すること

- [ ] Vercelにデプロイされているか
- [ ] 閲覧専用モードが正しく動作するか
- [ ] 「レビュー完了」ボタンでメール送信が動作するか
- [ ] 動画が正しく表示されるか（ローカルでダウンロード済みの動画）

## 🚀 デプロイ方法

1. ローカルで変更をコミット
2. 「☁️ Vercelにデプロイ」ボタンをクリック
3. または、GitHubにプッシュ（自動デプロイ）

## 💡 便利なTips

- **ブックマーク**: クライアントさんにブックマークしてもらうと便利
- **ショートカット**: iPhoneのホーム画面に追加してもらうとアプリのように使える
- **更新通知**: 新しい候補者が追加されたら、LINEで「更新しました」と連絡
