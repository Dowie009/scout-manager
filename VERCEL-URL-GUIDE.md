# 🔗 VercelのURL確認ガイド

## 📍 VercelのURLを確認する方法

### 1. Vercelダッシュボードで確認

1. **Vercelダッシュボードを開く**
   - https://vercel.com/dashboard
   - ログインする

2. **プロジェクトを選択**
   - 「スカウト管理アプリ」のプロジェクトをクリック

3. **URLを確認**
   - 「Visit」ボタンの横に表示されているURL
   - 例: `https://scout-manager-xxxxx.vercel.app`
   - または、デプロイ履歴から最新のデプロイを確認

### 2. カスタムドメインを設定（オプション）

より分かりやすいURLにしたい場合：

1. Vercelダッシュボード → プロジェクト → **Settings**
2. **Domains** を選択
3. カスタムドメインを追加
   - 例: `scout.yourdomain.com`

## ⚠️ Not Foundエラーの原因

### 考えられる原因

1. **まだデプロイされていない**
   - GitHubにプッシュしていない
   - Vercelでデプロイが失敗している

2. **間違ったURLを使用している**
   - 古いURLを使用している
   - タイプミス

3. **プロジェクトが削除されている**
   - Vercelでプロジェクトが削除された

### 解決方法

1. **デプロイを確認**
   ```bash
   # ローカルで確認
   cd /Users/dowie009/scout-manager
   git status
   git push
   ```

2. **Vercelでデプロイ状況を確認**
   - Vercelダッシュボード → Deployments
   - 最新のデプロイが成功しているか確認

3. **正しいURLを確認**
   - Vercelダッシュボードで最新のURLを確認

## 🚀 デプロイ手順

1. **ローカルで変更をコミット**
   ```bash
   cd /Users/dowie009/scout-manager
   git add -A
   git commit -m "Update"
   git push
   ```

2. **Vercelで自動デプロイ**
   - GitHubにプッシュすると自動的にデプロイが開始
   - 通常1-2分で完了

3. **デプロイ完了後、URLを確認**
   - Vercelダッシュボードで「Visit」ボタンをクリック
   - URLをコピー
